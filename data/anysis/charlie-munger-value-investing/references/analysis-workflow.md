# 分析流程详解

数据齐备后执行。倍数与买入区见 `valuation-framework.md`；思维框架见 `mental-models.md`。

## 步骤 1：行业估值体系

不同生意不能共用同一 PE 尺子。行业参考区间见 `valuation-framework.md`。

**原则：** PE 低不一定便宜，PE 高不一定贵。必须绑定增长、ROE、现金流、行业地位。

## 步骤 2：是不是好公司

### 优质消费品牌常见特征

- 毛利率高  
- 净利率稳定  
- ROE 长期较高  
- 经营现金流多年 ≥ 或接近净利润  
- 品牌有复购  
- 不严重依赖单一爆品  
- 销售费用率未失控  
- 存货与应收未明显恶化  
- 能持续分红或回购  

### 快速评分表

| 维度 | 好 | 一般 | 差 |
|------|----|------|-----|
| 收入增长 | 连续增长 | 增速放缓 | 连续下滑 |
| 利润增长 | 快于或接近收入 | 小幅波动 | 明显下滑 |
| 毛利率 | 稳定或提升 | 小幅波动 | 持续下降 |
| 净利率 | 稳定 | 波动 | 被费用吞噬 |
| ROE | ≥ 15% | 8%–15% | < 8% |
| 现金流 | 长期好于利润 | 接近利润 | 长期差于利润 |
| 费用率 | 可控 | 偶有上升 | 持续失控 |
| 资产负债表 | 现金充足 | 中性 | 高负债、高存货 |

## 步骤 3：所处阶段

阶段决定估值倍数。

| 阶段 | 特征 | 估值处理 |
|------|------|----------|
| 高成长 | 收入利润双高增 | 可给较高 PE |
| 成熟稳定 | 增速中低，现金流好 | 行业平均 PE |
| 增长换挡 | 增速下滑但质量仍好 | 低于历史均值 PE |
| 困境修复 | 利润低点，待改善 | PE 可能失真 → PB/PS/现金流 |
| 衰退 | 收入利润长期下滑 | 不能只看低 PE |

**关键分叉：**

- 短期调整 + 特许经营权仍在 → 低估值可能是机会。  
- 盈利中枢永久下移 → 低估值可能是价值陷阱。  

## 步骤 4：财报质量

### 利润表

看：收入、毛利率、销售费用率、净利率侵蚀、扣非 vs 归母。

**风险信号：** 收入增利润不增；利润靠非经常性；销售费用率飙升；毛利率下滑（竞争或促销）。

### 现金流量表

看：经营现金流 vs 净利润多年关系；应收、存货是否暴增；自由现金流是否稳定。

| 形态 | 解读 |
|------|------|
| 利润↑且经营现金流↑ | 质量较好 |
| 利润↑但经营现金流↓ | 需警惕 |
| 多年现金流显著差于利润 | 利润质量存疑 |

### 资产负债表

看：货币资金、有息负债、存货、应收、商誉、资产负债率。

**风险信号：** 存货增速 >> 收入；应收增速 >> 收入；商誉占净资产过高；有息负债持续上升。

## 步骤 5：历史估值

| 指标 | 作用 |
|------|------|
| PE TTM | 对过去盈利的定价 |
| Forward PE | 对预期盈利的定价 |
| PB | 是否靠近资产价值 |
| PS | 利润波动或亏损时 |
| 股息率 | 下行保护粗估 |
| 历史 PE 分位 | 相对自身历史贵贱 |

| 历史分位 | 含义 |
|----------|------|
| 0%–10% | 历史极低，值得重点研究 |
| 10%–30% | 偏低，可能有机会 |
| 30%–70% | 中性 |
| 70%–90% | 偏贵 |
| 90%–100% | 历史高位，谨慎追高 |

历史低估 ≠ 内在便宜。须确认基本面未永久恶化。

## 步骤 6：同业对比

### A 股 / 本地同行

至少比：龙头、同规模、同定位、同阶段公司。

指标：PE、PS、PB、ROE、毛利率、净利率、收入增速、利润增速。

### 海外可比（定锚，不照搬）

| 行业 | 示例 |
|------|------|
| 美妆护肤 | Estée Lauder、L'Oréal、Shiseido、e.l.f.、Ulta、Coty |
| 鞋服 | Nike、Adidas、Deckers、Steve Madden、Crocs |
| 奢侈品 | LVMH、Hermès、Kering、Richemont |
| 零售 | Ulta / Sephora 所属集团、Target、Walmart 等 |

目的：观察成熟 / 高成长 / 困境品牌分别给多少估值。须调整市场、流动性、增长、治理差异。

## 步骤 7–8：三情景与买入区

见 `valuation-framework.md`。

## 步骤 9：可提高买入价的条件

仅在出现真实改善信号后，才上调合理 PE：

- 营收重新转正增长  
- 扣非净利润恢复增长  
- 毛利率稳定或提升  
- 销售费用率下降  
- 经营现金流恢复  
- 新品牌 / 产品 / 市场贡献真实利润  
- 回购、分红增强股东回报  
- 行业景气回升  

## 步骤 10：否决与风险触发

买入前必须写清「什么情况下我错了」。

### 基本面否决

- 连续两个季度收入下滑  
- 扣非下滑幅度扩大  
- 毛利率明显下降  
- 销售费用率持续上升  
- 经营现金流显著差于净利润  
- 存货或应收异常增加  
- 主品牌老化，第二曲线失败  
- 管理层频繁变动  
- 大额商誉减值风险  
- 价格战 / 竞争恶化  

### 估值否决

- 股价已超过乐观情景  
- 估值高于可比但增长不占优  
- 上涨主要靠情绪而非业绩  
- 盈利预测被持续下修
