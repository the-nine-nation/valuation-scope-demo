# 数据收集清单

评分与估值前必须尽量收齐。优先年报 + 最近一期季报；历史跨度 3–5 年。

## 行情 / 估值

| 数据项 | 用途 |
|--------|------|
| 当前股价 | 市值、估值 |
| 总股本 / 流通股本 | 市值、每股收益 |
| 总市值 | 体量 |
| PE TTM | 盈利估值 |
| Forward PE | 有一致预期时 |
| PB | 资产底、深跌辅助 |
| PS | 利润失真或亏损时 |
| 股息率 | 股东回报 / 下行缓冲 |
| 52 周高低点 | 市场位置 |
| 历史估值高低 | 相对自身历史贵贱 |

## 财务（近 3–5 年 + 最近季度）

| 指标 | 重点看什么 |
|------|------------|
| 营业收入 | 是否增长、是否放缓 |
| 归母净利润 | 利润路径 |
| 扣非净利润 | 真实盈利 |
| 毛利率 | 品牌力、定价权、结构 |
| 净利率 | 费用与模式质量 |
| 销售费用率 | 消费品牌关键：是否「砸钱买增长」 |
| 管理费用率 | 组织效率 |
| 研发费用率 | 产品力、长期投入 |
| 经营现金流 | 利润是否扎实 |
| 自由现金流 | 分红、回购、再投资能力 |
| ROE | 股东回报质量 |
| 存货周转 | 消费、鞋服、美妆尤甚 |
| 应收账款 | 渠道与回款 |
| 资产负债率 | 财务风险 |

## 业务结构

| 维度 | 要回答的问题 |
|------|----------------|
| 品牌矩阵 | 是否单品牌依赖？有无第二曲线？ |
| 产品结构 | 护肤/彩妆/鞋服/零售/代工等占比 |
| 渠道结构 | 线上/线下/经销/直营/平台 |
| 地区结构 | 国内/海外 |
| 价格带 | 大众/中高端/高端/奢侈 |
| 爆品能力 | 能否持续打造爆品 |
| 复购属性 | 高频还是低频 |
| 竞争对手 | 国内外直接对手 |

## 采集规则

1. 关键数字写明 **截止日期** 与 **来源**。  
2. 优先交易所 / 公司公告，其次正规数据终端。  
3. 缺失字段标 `暂缺`，并说明对结论置信度的影响。  
4. 多品牌集团尽量拿到分部收入（及披露的利润贡献）。
