# 估值框架

行业 PE、公式、三情景、分层买入区、分档建仓。

## 行业 PE 参考区间

| 类型 | 合理 PE | 说明 |
|------|--------:|------|
| 成熟鞋服品牌 | 10–18 倍 | 增长慢、周期性较强 |
| 稳定消费品牌 | 15–25 倍 | 现金流好、品牌稳定 |
| 美妆护肤品牌 | 18–30 倍 | 毛利率高、品牌弹性大 |
| 高成长美妆 / 彩妆 | 30–50 倍 | 必须有高增长支撑 |
| 奢侈品龙头 | 25–40 倍 | 全球化、强定价权 |
| 零售渠道公司 | 12–25 倍 | 取决于同店与库存周期 |
| 亏损或利润剧烈波动 | PE 失真 | 重点看 PS、PB、现金流与拐点 |

结合增长可持续性、ROE、自由现金流转化、竞争地位、周期位置上下调整。

## 核心公式

```text
预期 EPS = 预期归母净利润 / 总股本
合理股价 = 预期 EPS × 合理 PE
```

或：

```text
合理市值 = 预期归母净利润 × 合理 PE
合理股价 = 合理市值 / 总股本
```

PE 失效时（亏损、利润谷底）：

```text
合理价 ≈ 自由现金流 / 要求收益率
或 PB 底 × 每股净资产
或 PS × 常态化收入（并给出利润率路径）
```

## 三情景估值（必做）

未来 EPS / 净利润可参考：历史、TTM、一致预期、自建保守预测。

| 情景 | 假设 | PE 立场 | 价格 |
|------|------|---------|------|
| 悲观 | 收入仍弱、利润承压 | 低估值带 | EPS × 低 PE |
| 中性 | 收入企稳、利润小幅恢复 | 中性带 | EPS × 中性 PE |
| 乐观 | 增长恢复、多品牌放量 | 偏高带 | EPS × 较高 PE |

### 示例

| 情景 | EPS | PE | 目标价 |
|------|----:|---:|-------:|
| 悲观 | 3.8 元 | 13 倍 | 49.4 元 |
| 中性 | 4.1 元 | 16 倍 | 65.6 元 |
| 乐观 | 4.5 元 | 20 倍 | 90.0 元 |

每条假设写清楚（增速、利润率、倍数）。悲观情景倾向低估 EPS（反过来想，留安全边际）。

## 分层买入区间

禁止只给单一目标价。

| 区间 | 对应估值 | 操作思路 |
|------|----------|----------|
| 深度低估区 | 悲观估值以下 | 特许经营权未坏 → 重点研究 / 分批 |
| 舒适买入区 | 悲观与中性之间 | 适合分批建仓 |
| 合理偏低区 | 接近中性 | 可小仓观察 |
| 合理区 | 接近行业均值 | 等业绩确认 |
| 偏贵区 | 超过乐观估值 | 不宜追高 |

### 分档建仓示例（仅示例）

| 档位 | 条件 | 仓位示例 |
|------|------|---------:|
| 第一档 | 合理偏低 | 20% |
| 第二档 | 舒适买入 | 30% |
| 第三档 | 深度低估 | 30% |
| 第四档 | 基本面确认反转 | 20% |

仓位比例不可替代个人风险承受能力与流动性约束。

## 增长与倍数匹配

| 盈利路径 | 倍数纪律 |
|----------|----------|
| 高且可持续增长 | 上沿倍数须有 ROE/现金支撑 |
| 稳定低增长 | 中下沿；更看股息与 FCF 收益率 |
| 增长换挡 | 相对历史下调倍数；勿盲目均值回归 |
| 困境修复 | 未验证前限制上行；用 PB/FCF 托底 |
| 衰退 | 多数情况不买；低 PE 不是理由 |

## 发布区间前的自检

1. **现价**已定价了怎样的增长与利润率？  
2. 若 **3 年零增长**，现价是否仍可接受（现金收益 / PB）？  
3. 乐观情景是否依赖英雄假设？  
4. 区间是否与同业、历史表一致？  
5. 若估值永不修复，是否仍愿做这家公司的股东？
