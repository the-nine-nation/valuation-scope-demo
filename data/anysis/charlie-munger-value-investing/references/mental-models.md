# 芒格心智模型专章

分析任何标的前，先用本清单做「思维消毒」。与估值步骤互补：这里管**怎么想**，其他文件管**怎么算**。

## 1. 能力圈（Circle of Competence）

| 检查项 | 通过标准 |
|--------|----------|
| 商业模式一句话能否说清 | 能说明「谁付钱、为何付钱、钱如何留下」 |
| 关键驱动变量是否可知 | 能列出 3 个以内决定盈亏的变量 |
| 竞争格局是否可跟踪 | 知道前 3 名对手及差异 |
| 会计是否可理解 | 收入确认、存货、渠道返利等不黑箱 |
| 若 5 年不看盘能否安心持有 | 靠生意本身而非题材 |

**规则：** 能力圈外 → 明确写「不在能力圈，仅作框架演练 / 拒绝给出买入区」；不得伪装成完整结论。

## 2. 护城河与质量

优先问「这生意难不难被抢走」，再问「现价贵不贵」。

| 护城河类型 | 消费品常见表现 | 证伪信号 |
|------------|----------------|----------|
| 品牌 / 心智 | 溢价、复购、低促销依赖 | 持续打折才能动销 |
| 成本 / 规模 | 渠道与供应链优势 | 费用率被迫抬升 |
| 转换成本 | 配方、习惯、生态 | 用户零成本切换 |
| 网络 / 渠道 | 网点密度、货架权 | 渠道集中流失 |
| 特许 / 监管 | 牌照、标准 | 政策松绑后竞争涌入 |

质量底线（与 `analysis-workflow.md` 评分表一致）：高毛利、稳净利、高 ROE、OCF≈利润、费用可控。

## 3. 安全边际（Margin of Safety）

- 用**悲观情景**定价，不用乐观故事定价。  
- 买入区应落在「熊市假设仍不亏逻辑」附近。  
- 不确定性越大（周期、转型、商誉、治理）→ 要求越厚的折扣。  
- 安全边际来自**价格 vs 保守价值**，不是来自「已经跌了很多」。

## 4. 反过来想（Inversion）

买入前强制完成：

1. 列出「什么情况证明我错了」（→ 否决条件）。  
2. 列出「低 PE 为何仍是陷阱」（→ 价值陷阱清单）。  
3. 问：若未来 3 年零增长，当前价格是否仍可接受？  
4. 问：若只能永远持有、不能交易，是否仍愿以现价成为股东？

### 价值陷阱速查

- 盈利中枢永久下移仍用旧 PE  
- 现金流长期差于利润  
- 行业结构性衰退  
- 单一爆品依赖且老化  
- 渠道压货制造收入  
- 靠补贴 / 非经常性损益「好看」

## 5. 误判心理学（投资中最常见的）

分析时主动扫描是否中招：

| 倾向 | 在投研中的样子 | 对冲动作 |
|------|----------------|----------|
| 锚定 | 死盯历史高价 / 历史高 PE | 重估盈利中枢后再比估值 |
| 社会认同 | 「机构都看好」 | 只看报表与竞争事实 |
| 确认偏误 | 只搜集看多证据 | 先写空方报告再综合 |
| 损失厌恶 | 套牢后平均成本心态加仓 | 用现价 vs 价值，忘掉成本 |
| 过度自信 | 点预测精确到分 | 给区间 + 情景，不给伪精确 |
| 可得性启发 | 被近期新闻带节奏 | 拉 3–5 年序列 |
| 沉没成本 | 「研究了很久必须下结论买入」 | 允许「观望 / 放弃」为合法输出 |
| 叙事诱惑 | 好故事替代差数字 | 故事必须映射到收入/利润/现金 |

## 6. 多元思维模型（简表）

不必每次全用，按行业点名 2–3 个：

| 模型 | 提问 |
|------|------|
| 微观经济学 | 定价权？单位经济？规模报酬？ |
| 竞争战略 | 五力中谁在吃利润？ |
| 会计 | 利润能否变成现金？ |
| 概率 | 情景权重是否诚实？ |
| 激励 | 管理层考核与股东是否一致？ |
| 复利 | ROE 与再投资回报能否长期 > 资本成本？ |
| 冗余 / 安全工程 | 负债、存货、商誉有无「单点故障」？ |

## 7. 机会成本与「太难」格

- 每个标的都要和「次优选择」比：同样资金买指数 / 更懂的好公司是否更优。  
- 芒格「too hard」格：信息不足、模式多变、依赖宏观猜顶时，**输出应为放弃或观望**，不是硬凑买入区。

## 8. 分析前 8 问（心智预检）

1. 这门生意我是否真懂？  
2. 护城河 5 年后是否大概率还在？  
3. 管理层激励是否靠谱？  
4. 当前价格隐含了怎样的增长与利润率？  
5. 我的熊市假设是否足够丑？  
6. 我因何可能在自欺（对照误判表）？  
7. 不买的机会成本是什么？  
8. 若必须反过来想，最可能的致命伤是什么？

全部通过后再进入数据收集与估值流程。
